# 局域网文件传输系统 - 傻瓜式操作指南

## 一、软件简介

这是一个简单的文件传输工具，可以让你在同一局域网内的手机和电脑之间互相传输文件，无需安装任何APP，只需使用浏览器即可。

## 二、准备工作

1. **确保电脑已安装Python**
   - 如果你的电脑还没有Python，先安装它：
     - 打开 `shared_files` 文件夹
     - 双击运行 `python-3.12.8-amd64.exe`
     - 安装时请勾选 "Add Python to PATH"（添加Python到环境变量）
     - 然后点击 "Install Now" 完成安装

2. **确保手机和电脑在同一WiFi下**
   - 手机和电脑必须连接到同一个WiFi网络才能互相传输文件

## 三、启动服务

### 方法：直接双击运行批处理文件

1. 在项目文件夹中找到名为 `启动服务器.bat` 的文件
2. 双击它，会弹出一个黑色的命令窗口
3. 等待窗口中显示 "按任意键启动服务器..."
4. 按键盘上的任意键（回车键）
5. 看到 "服务器启动，访问地址: http://0.0.0.0:5000" 就表示成功了

## 四、使用方法

### 1. 查看你的电脑IP地址

在弹出的黑色命令窗口中，找到类似这样的一行：
```
   IPv4 地址 . . . . . . . . . . . . : 192.168.1.100
```

记下这个IP地址（例如：192.168.1.100）

### 2. 在手机上访问

1. 打开手机浏览器（随便哪个浏览器都可以，如Chrome、Safari、微信内置浏览器）
2. 在地址栏输入：`http://你的电脑IP:5000`（例如：http://192.168.1.100:5000）
3. 按下回车键，即可看到文件传输界面

### 3. 上传文件（手机 → 电脑）

1. 在手机浏览器中，选择 "手机 → 电脑" 标签
2. 点击 "选择文件" 按钮
3. 选择你要发送的文件（可以多选）
4. 点击 "开始上传" 按钮
5. 等待上传完成，会显示 "成功上传 X 个文件"
6. 上传的文件会保存在电脑上的 `received_files` 文件夹中

### 4. 下载文件（电脑 → 手机）

1. 首先，把你要分享给手机的文件复制到电脑上的 `shared_files` 文件夹中
2. 在手机浏览器中，选择 "电脑 → 手机" 标签
3. 点击 "刷新文件列表" 按钮
4. 看到你要下载的文件后，点击右侧的 "下载" 按钮
5. 等待下载完成，文件会保存到手机的默认下载目录

## 五、停止服务

当你不需要使用文件传输服务时：
1. 回到电脑上的黑色命令窗口
2. 按键盘上的 `Ctrl + C` 组合键
3. 命令窗口会关闭，服务就停止了

## 六、常见问题及解决方法

### 问题1：双击 `启动服务器.bat` 后窗口一闪而过
- 解决方法：检查是否已正确安装Python，并且勾选了 "Add Python to PATH"

### 问题2：手机浏览器无法访问服务器
- 解决方法：
  1. 确保手机和电脑连接到同一个WiFi网络
  2. 检查电脑IP地址是否输入正确
  3. 检查服务器是否已经成功启动

### 问题3：上传文件失败
- 解决方法：
  1. 检查文件大小，不要上传过大的文件
  2. 检查网络连接是否稳定
  3. 尝试重新启动服务器

### 问题4：找不到上传/下载的文件
- 上传的文件在电脑的 `received_files` 文件夹中
- 要分享的文件需要放到电脑的 `shared_files` 文件夹中

## 七、注意事项

1. 这个工具仅在同一局域网内使用，不能用于跨网络传输
2. 传输过程中请保持电脑和手机的网络连接稳定
3. 使用完毕后记得关闭服务器，节省电脑资源
4. 定期清理 `received_files` 和 `shared_files` 文件夹，避免占用过多磁盘空间
